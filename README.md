AI Agent 智能助手项目
一个基于 LangChain 和 DeepSeek 的多智能体协作系统，支持实时搜索、数学计算、时间查询等功能，具备短期记忆和混合任务处理能力。

📋 项目概述
本项目实现了一个多智能体协作框架，通过主管 Agent 统一调度多个专业 Agent，能够智能拆解并执行用户的复杂混合指令。系统具备以下核心能力：

🔍 实时搜索：集成 Tavily 搜索引擎，获取最新信息

🧮 数学计算：支持复杂表达式计算

⏰ 时间查询：实时获取系统时间

💬 短期记忆：同一会话内保持上下文连贯

🔀 混合任务处理：自动拆解多意图问题（如“现在几点了？顺便查一下今天的AI新闻”）

工作流程
用户输入 → 主管 Agent

任务分析：主管识别问题类型（time/calc/search/none）

子问题提取：为每个任务提取对应的子问题

并行/串行执行：各专员 Agent 独立执行任务

结果整合：主管汇总所有结果，生成连贯回答

记忆存储：对话存入短期记忆，供后续使用

✨ 核心功能
1. 多意图识别与拆解
python
输入: "现在几点了？顺便查一下今天的AI新闻，再算一下15*37"
输出: 
📋 识别到任务: time, search, calc
🔧 正在执行: time → "当前时间: 15:30"
🔧 正在执行: search → "AI新闻: OpenAI发布新模型..."
🔧 正在执行: calc → "计算结果: 555"
🤖 最终回答: "现在是15:30。今天的AI新闻有...另外，15*37的结果是555。"
2. 实时信息搜索
集成 Tavily 搜索引擎，支持获取最新资讯，搜索结果自动清洗去噪。

3. 短期对话记忆
基于 LangGraph 的 MemorySaver，同一会话内保持上下文连贯，支持追问。

4. 可扩展的工具集
模块化设计，可轻松添加新的工具和对应的专员 Agent。
